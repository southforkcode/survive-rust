# Survival

## Setup

To set up the required git hooks for this project, run the following command after cloning the repository:

```bash
git config core.hooksPath .githooks
```

## Game Overview

- Game loop is day-by-day, where the player must make decisions to survive.
- Each day the player must decide on a specific action to take like gathering food, water, fuel, or building out their camp-site.
- This is currently a text-based game, but I would like to add a GUI later on.
